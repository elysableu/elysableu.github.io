HTML & CSS Cheatsheet

HTML References

Tag Name Description
<html> HTML Marks the beginning of HTML code
<head> Head Container for important doucment information (i.e. page, links to stylesheets, metadata)
<title> Title Sets title of page
<body> Body Contains all the content to be dispalyed on page
<h1> - <h6> Heading Creates headings from size h1 to h6
<p> Paragraph Creates a paragraph of text
<a> Anchor Creates a clickable hyperlink for navigation
<img> Image Displays an image on page, use src
<style> Style Add CSS styles to HTML code; Typically placed within <head> tag
<form> Form Creates a container for user input; such forms as input fields, radio buttons, checkboxes, etc.
<table> Table Create tables on your page to organize data
<div> Div Create a container for content on the page; group elements together and apply CSS styles to them
<!-- --> Commment Add comments to the code or temporarily disable certain sections

CSS References

Tag Name Description
color Color Set text color
font-size Font-size Set the font size
background-color Background-color Set the background color
background-image Background-image Set the background color
margin Margin Create space around an element
padding Padding Create space within an element
border Border Create a border around an element
display Display Specify the display behavior of an element
position Position Set the position of an element
transform Transform Transform an element by rotating, scaling, or translating
